# Node URL Handler
Node URL Handler help you to handle URL as strings for NodeJS. Once you have a URL stored as string, you can use this Module to get:
 - get/set Protocol
 - get/set Subdomain
 - get/set Domain
 - get/set Path
 - get/set Query
 - get/set Anchors
 - get Full URL
